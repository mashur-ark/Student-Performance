Project Description

This project aims to develop a machine learning model that predicts a student's academic performance based on various demographic, social, and educational factors. By analyzing the relationships between these factors and student grades, the model can assist educators in identifying students who may require additional academic support. The project also compares different machine learning algorithms to determine which model provides the highest prediction accuracy.

Goal / Objectives of the Project

The primary objectives of this project are:

To analyze the factors that influence student academic performance.
To clean and preprocess the student performance dataset for machine learning.
To build predictive machine learning models for student performance.
To compare the performance of different machine learning algorithms.
To identify the best-performing model using evaluation metrics.
To provide insights that can help improve educational decision-making.

Dataset Description

The dataset contains information about Portuguese secondary school students, including demographic information, family background, study habits, and academic records.

There are two datasets:

student-mat.csv (Mathematics)
student-por.csv (Portuguese Language)

For this project, the student-mat.csv dataset is used.

Number of instances: 395 students

Number of attributes: 33 features
